# NordVPN commands

The following commands are supplied:

- login.sh
- logout.sh
- config.sh
- connect.sh
- list_connected
- exit-node.sh
- status.sh

The following devices run NordVPN:

- DELL - DellNord
- HP - HPNord
- Pixel
- Tab8
- sunndal
- raspberry

The raspberry will be configured as an exit-node allowing the other devices to 
route all traffic through this device.
